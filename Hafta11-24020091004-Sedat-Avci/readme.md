# Nesne Yönelimli Programlama – Hafta 11 (Bölüm 12) | İlk 10 Sorunun Çözümleri

Bu repo, **Nesne Yönelimli Programlama** dersi kapsamında hazırladığım **Hafta 11** ödevindeki **10 sorunun** çözümlerini içerir.  
Odak: **Exception Handling (try-catch)**, **girdi doğrulama** ve **custom exception** kullanımı.

- **Bolum12-4** → `Loan`: geçersiz parametrelerde `IllegalArgumentException`
